# IT Automation Portfolio

Este portafolio muestra proyectos de automatización IT enfocados en eficiencia, reducción de errores y mejora de procesos.

## Tecnologías
- Power Automate / Power Platform
- Power Automate Desktop
- SharePoint Online
- Excel Online
- Docker (en proyectos futuros)

## Estructura
- `power-automate/` → proyectos de Power Automate
- `power-automate-desktop/` → proyectos RPA
- `docker/` → proyectos de infraestructura simple
